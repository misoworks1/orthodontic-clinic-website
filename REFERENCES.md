## 프론트엔드 범주별 상세 분석

### 범주 1: 풀스크린 히어로 + 스크롤 스토리텔링형
**대표 사이트:** 서울클리어 본사, 대전청담수치과, 마인드교정치과, 강남새로치과

**기술적 특징:**
- GSAP ScrollTrigger로 스크롤 위치 기반 애니메이션 트리거
- AOS(Animate On Scroll) 라이브러리로 요소 진입 애니메이션
- WOW.js로 스크롤 진입 시 CSS 애니메이션 활성화
- Headroom.js로 스크롤 방향 감지 네비게이션 자동 숨김/표시
- IntersectionObserver API로 성능 최적화
- 섹션별 모듈 구조로 스토리 전개

**적합한 경우:** 브랜드 스토리, 차별화 메시지를 강조하고 싶을 때. 프리미엄 이미지 필요 시.

---

### 범주 2: 이미지 슬라이더/캐러셀 히어로형
**대표 사이트:** 스마일디치과, 아인스치과, 청아치과, 예바치과, 서울특별치과교정과

**기술적 특징:**
- Swiper.js가 가장 보편적 (자동재생, 페이지네이션, 네비게이션)
- Slick Carousel이 두 번째로 많이 사용
- Owl Carousel은 imweb/Doz 빌더와 함께 사용
- BxSlider는 레거시 사이트에서 발견
- 슬라이드 간격: 1.8초~8초 (평균 3~5초)
- 전환 효과: 페이드, 슬라이드, 큐브 등

**적합한 경우:** 여러 서비스/이벤트를 동시에 노출하고 싶을 때. 가장 무난한 선택.

---

### 범주 3: 미니멀리즘/타이포그래피 중심형
**대표 사이트:** 서울D교정치과, BFO교정치과

**기술적 특징:**
- 세리프 폰트(나눔명조, Sorts Mill Goudy, Cormorant) 활용
- 풍부한 화이트스페이스
- 제한적 컬러 팔레트 (2~3색)
- 과도한 애니메이션 배제
- 포트폴리오/갤러리 중심 콘텐츠

**적합한 경우:** 원장 개인 브랜딩, 소규모 1인 교정치과. 고급스러움 강조.

---

### 범주 4: 카드 레이아웃/정보 밀집형
**대표 사이트:** 아이디치과, CS치과병원

**기술적 특징:**
- 비대칭 카드 그리드 레이아웃
- 서비스별 카테고리 분류가 명확
- 다단계 드롭다운 네비게이션
- JWPlayer 등 동영상 플레이어 통합
- 별도 모바일 사이트(/m/) 운영하는 경우 있음

**적합한 경우:** 종합 치과, 다양한 진료 과목을 보유한 대형 병원.

---

### 범주 5: WordPress/Elementor 기반 모던형
**대표 사이트:** 청아치과, BFO교정치과, 서울USC치과, 마인드교정치과

**기술적 특징:**
- WordPress CMS + Elementor Pro 페이지빌더
- Astra, Nectar 등 프리미엄 테마 활용
- Revolution Slider, Swiper 등 슬라이더 플러그인
- KBoard(게시판), WP Rocket(최적화) 등 플러그인 생태계
- 3~4단계 반응형 브레이크포인트
- Google Fonts 통합 (Montserrat, Noto Sans 등)

**적합한 경우:** 중소규모 치과의 가장 현실적인 선택. 제작 속도와 관리 용이성 균형.

---

### 범주 6: 웹빌더/SaaS 플랫폼형
**대표 사이트:** 연세올곧은치과(Wix), CS치과병원(imweb), 서울바른 인천(modoo)

**기술적 특징:**
- Wix: CSS Grid 기반, View Transition API, Thunderbolt 렌더링 엔진
- imweb: Owl Carousel 통합, 템플릿 기반 빌더
- modoo: 네이버 통합, 가장 간편하지만 커스터마이징 제한
- 코드 접근성이 제한적

**적합한 경우:** 빠른 런칭, 최소 비용, 기술 인력 없는 소규모 치과.

---

### 범주 7: Bootstrap 기반 클래식형
**대표 사이트:** 서울D교정치과, 하하치과

**기술적 특징:**
- Bootstrap 3/4 그리드 시스템 (12컬럼)
- jQuery 기반 인터랙션
- 표준 브레이크포인트 (768/992/1200px)
- AOS 등 경량 애니메이션 라이브러리 추가
- 안정적이지만 디자인 차별화 어려움

**적합한 경우:** 안정성 중시, 레거시 유지보수 편의.

## 공통 프론트엔드 패턴 요약

### 필수 기능 (90% 이상 채택)
- **고정 헤더 네비게이션**: 스크롤 시 상단 고정
- **모바일 햄버거 메뉴**: 반응형 필수
- **온라인 상담/예약**: 네이버예약, 카카오톡, 전화
- **의료진 프로필 섹션**: 학력/경력/전문 분야

### 보편적 기능 (60~90%)
- **Before/After 갤러리**: 교정 치과 핵심 콘텐츠
- **치료 후기/리뷰 섹션**: 신뢰도 구축
- **네이버 지도 연동**: 오시는 길
- **우측/하단 고정 퀵메뉴**: 전화, 카톡, 예약 바로가기

### 트렌드 기능 (30~60%)
- **스크롤 애니메이션**: AOS, GSAP, WOW.js
- **풀스크린 비디오 배경**: 프리미엄 이미지
- **다국어 지원**: GTranslate 등
- **SNS 연동**: 인스타그램, 유튜브, 블로그
