### 우리 사이트에 적용 가능한 점

### A. 디자인 트렌드 종합

| 요소 | 주류 트렌드 | 추천 적용 |

| **히어로** | 비디오 또는 대형 타이포 | "진단이 90%" 대형 타이포 히어로 |
| **애니메이션** | GSAP/AOS 스크롤 트리거 | 텍스트 리빌 + 클립패스 마스크 |
| **증례** | 갤러리/슬라이더/탭 분류 | 드래그 Before/After + 카테고리 탭 |

### B. 박주환 원장 맞춤 핵심 전략

1. **"진단이 90%" 히어로** — 본사와 일관된 메시지, 부산서면점 차별화 서브카피 추가
2. **서울대 교정과 전문의 스펙** — 숫자/통계로 희소성 어필 (전체 치과의사 중 교정과 전문의 비율)
3. **세리프 + 산세리프 혼합** — 학술적 권위와 현대적 세련미 동시 표현
4. **증례 갤러리** — 드래그 비교 슬라이더 + 케이스 분류 체계 + 치료 기간 표시
5. **텍스트 리빌 애니메이션** — 과하지 않은 미니멀 인터랙션
6. **정직한 카피 톤** — "함부로 약속하지 않는다" / "진단이 다르니 결과가 다릅니다"
7. **4단계 진료 프로세스** — 진단 중심 워크플로우 시각화

### D. 최종 디자인 방향 제안

**서울클리어 부산서면점 = "진단의 과학"을 미니멀하게 표현하는 프리미엄 교정 전문 사이트**

- **톤:** 본사 "진단이 90%"를 계승하되, 박주환 원장의 서울대 학술적 깊이를 가미
- **컬러:** 화이트 + 딥네이비 (#0c1531 계열) + 웜 베이지 액센트
- **폰트:** Pretendard 본문 + Nanum Myeongjo 또는 유사 세리프 헤드라인
- **히어로:** "교정치료는 진단이 90%입니다" 대형 세리프 타이포 + 미니멀 배경
- **핵심 섹션:** 진단 프로세스 → 원장 스펙 → 증례 갤러리 → 상담 CTA
- **애니메이션:** 텍스트 리빌 (CSS clip-path) + AOS 스크롤 페이드 + 드래그 B/A
- **증례:** 카테고리별 탭 분류 + 드래그 비교 + 치료 기간/설명 포함
